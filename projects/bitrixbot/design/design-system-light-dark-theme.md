## Scrollbar — как применять

```css
/* WebKit */
::-webkit-scrollbar {
  width: 10px;
  height: 10px;
}

::-webkit-scrollbar-track {
  background: var(--scrollbar-track);
}

::-webkit-scrollbar-thumb {
  background: var(--scrollbar-thumb);
  border-radius: 999px;
  border: 2px solid var(--scrollbar-track);
}

::-webkit-scrollbar-thumb:hover {
  background: var(--scrollbar-thumb-hover);
}

::-webkit-scrollbar-thumb:active {
  background: var(--scrollbar-thumb-active);
}

::-webkit-scrollbar-corner {
  background: var(--scrollbar-corner);
}

/* Firefox */
* {
  scrollbar-width: thin;
  scrollbar-color: var(--scrollbar-thumb) var(--scrollbar-track);
}
```

---

## Рекомендации

- Для графиков используй только `chart-*`
- Для роста и падения всегда показывай цвет + стрелку + значение
- Для таблиц держи отдельные токены `table-*`, не смешивай их с карточками
- Для полей ввода и попапов лучше использовать собственные токены, а не просто `bg-card`
- Полосы прокрутки делай спокойными и нейтральными, чтобы они не перетягивали внимание
